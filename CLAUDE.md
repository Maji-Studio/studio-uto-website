## Agent skills

### Issue tracker

Issues and PRDs live as local markdown files under `.scratch/<feature>/` in this repo (no remote configured). See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical triage roles, each using its default string (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context — one `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
